📋 Rakuten API 발급 가이드 무료
01
라쿠텐 개발자 센터 접속
라쿠텐 웹서비스 개발자 페이지에 접속합니다. 일본어지만 Google 번역으로 쉽게 진행 가능합니다.
→ webservice.rakuten.co.jp
02
라쿠텐 계정 생성 / 로그인
라쿠텐 글로벌 계정으로 로그인합니다. 기존 라쿠텐 쇼핑 계정이 있으면 동일하게 사용 가능합니다.
→ 계정 만들기
03
앱 등록 → Application ID 발급
"アプリ登録" (앱 등록) 버튼 클릭 → 앱 이름, URL 등 입력 → 즉시 Application ID 발급됩니다.
→ 앱 등록 페이지
04
위 입력창에 키 붙여넣기
발급된 Application ID를 복사해서 상단 API Key 입력창에 붙여넣으면 실시간 데이터로 자동 전환됩니다.
→ 키 입력창으로 이동
